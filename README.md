## simple_security3
Form 기반을 json 데이터로 API 인증 제공

단, login 호출시에는 데이터형은 form형태로 보내야함.(이게 전부인데 ㅡㅡ;;) 그외의 응답은 성공/실패는 json 데이터 형으로 처리됨. 그렇기 때문에 로그인 응답에 대해 클라이언트 측에서 처리 가능.
